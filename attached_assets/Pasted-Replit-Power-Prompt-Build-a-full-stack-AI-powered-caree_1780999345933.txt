Replit Power Prompt

Build a full-stack AI-powered career recommendation web application called PathFinder 10k.

The product is designed for secondary school students in Nigeria, with an initial focus on Borno State and Maiduguri. It should help SSS1–SSS3 students and JAMB candidates choose suitable university career paths based on their interests, strengths, and assessment answers. The target is to support up to 10,000 students.

Tech Stack

Use:

React for the frontend
Node.js/Express for the backend
Supabase for authentication and database storage
Gemini API for AI-powered career explanations
A placeholder/mock ML prediction service for now, but structure the code so a real ML API can be connected later
